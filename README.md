# My React App

这是一个使用 React 和 TypeScript 构建的简单应用程序。

## 项目结构

```
my-react-app
├── src
│   ├── components
│   │   └── App.tsx       # 定义主要的 React 组件
│   ├── index.tsx         # 应用程序的入口点
│   └── styles
│       └── App.css       # 组件样式定义
├── public
│   ├── index.html         # HTML 模板
│   └── favicon.ico        # 应用程序图标
├── package.json           # npm 配置文件
├── tsconfig.json          # TypeScript 配置文件
└── README.md              # 项目文档
```

## 安装依赖

在项目根目录下运行以下命令以安装依赖：

```
npm install
```

## 启动应用

使用以下命令启动开发服务器：

```
npm start
```

然后在浏览器中访问 `http://localhost:3000` 查看应用程序。

## 构建应用

使用以下命令构建生产版本：

```
npm run build
```
